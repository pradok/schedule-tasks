## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Pre-requisites

  1. Dotenv cli
  2. Docker
  3. Node version 18 or above

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# integration tests
# dotenv needed for using env.test for integration tests
$ npm install -g dotenv-cli
$ npm run test:integration

# destroy docker test db container after running integration tests
$ npm run docker:down

# test coverage
$ npm run test:cov
```
